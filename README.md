# == Features ==
* HTML Sample template 同梱
* Google reCaptcha 付き(Keyを追加)
* メール送信の設定は、sendmail.php 内にて設定

# == Installation ==
* PHPMailerファイルのディレクトリー構成は、include/
* require_once('phpmailer/PHPMailerAutoload.php'); Path
